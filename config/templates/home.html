Login API

This API provides endpoints for user management, including creating accounts, authentication, and profile management.

Endpoints

POST /api/create-user/

Description: Create a new user account.

Request: JSON body with name, email, and password.

Response: Returns access and refresh tokens on success.

POST /api/token/

Description: Obtain access and refresh tokens by providing valid credentials.

Request: JSON body with email and password.

Response: Returns access and refresh tokens on success.

POST /api/logout/

Description: Log out a user by blacklisting their refresh token.

Request: JSON body with refresh token.

Response: Returns a success message on logout.

GET /api/profile/

Description: Retrieve the profile information of the authenticated user.

Request: Requires authentication with an access token.

Response: Returns user details.

POST /api/change-password/

Description: Change the password for the authenticated user.

Request: JSON body with old_password and new_password.

Response: Returns a success message on password change.

POST /api/refresh/

Description: Refresh the access token using a valid refresh token.

Request: JSON body with refresh token.

Response: Returns a new access token.
